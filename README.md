Es un programa de consola en C# para gestionar una biblioteca con libros, usuarios y préstamos.
Funciona mediante menús que permiten al usuario navegar y realizar acciones de forma sencilla.
Toda la información se guarda temporalmente en memoria mientras el programa está en ejecución.
El sistema controla préstamos y devoluciones verificando disponibilidad y existencia de datos.

Instrucciones de ejecución 
requisitos: tener git

-ir al repo; darle al boton verde que dice code, y copiar ese link

-en su pc, ir al directorio de preferencia, donde quiere terner el proyecto, abrir git y escribir

-git clone https://github.com/dimas528/DesafioFinal-BM260418-BE260699

-luego entrar a la carpeta "DesafioFinal-BM260418-BE260699"

-luego hacer git pull origin main, por cualquier cosa

-luego entrar en el proyecto que se quiere ejecutar y darle clic al Program.cs, y solo se abrira el vs.
